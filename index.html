About

Living the dream and coding like a barbarian! For the past several years, I've been on a journey to learn programming and become a professional developer. I originally began learning the basics of web design (HTML/CSS) which led to building several sites using the Wordpress framework. After spending about a year completing various online courses with Udemy, Codecademy and General Assembly This year, I started the FreeCodeCamp curriculum and I would have to say it's one of the most well thought out bootcamp I've seen.

Over the last five months, I've been building up my skills in JavaScript, JQuery, JSON and APIs. Within the next few weeks, I hope to have reached the point in the program where I will be able to claim my Front End Developement Certificate and start working in the field professionally.
